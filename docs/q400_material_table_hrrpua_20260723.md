# Q400材料表HRRPUA敏感性案例

本案例以正常HRRPUA Q400案例为对照，仅修改有材料表直接对应关系的HRRPUA。
固定参数为Q=400 J/cm2、W=100 kt、az=270 deg、el=15 deg、T_END=1500 s。
几何、网格、材料厚度、点燃温度、BURN_AWAY、探针及毁伤判据均不变。

| 对象 | HRRPUA (kW/m2) | 取值说明 |
|---|---:|---|
| RADM | 75 | 表中无直接对应，维持正常基准 |
| WINS | 250 | 表中无直接对应，维持正常基准 |
| BED | 400 | 尼龙织物 |
| CURT | 400 | 尼龙织物 |
| U4 | 350 | 驾驶舱仪表/电子舱 |
| SEAT | 450 | 聚氨酯泡沫 |
| H6 | 200 | 表中无PVC直接对应，维持正常基准 |
| H7 | 180 | 表中无CR橡胶直接对应，维持正常基准 |

该案例是材料表推荐值敏感性案例，不与高HRRPUA上限案例混合形成阈值结论。
