# U06 STL 实体体素填充与验证

## 目的

原 PyroSim 导出的 `U06.stl` 座椅在 FDS 中被表示为零厚度 `OBST` 壳面。该表示可用于显示外形和计算表面受热，但不能可靠表示聚氨酯泡沫的体积、质量、热惯性、持续燃烧和材料耗尽。

本次在不修改原 STL、原 FDS 和既有案例的前提下，建立独立的实体体素修正版。

## 几何处理

1. 将 `U06.stl` 按连通域拆分为 12 个部件。
2. 12 个部件分别均为封闭实体，原始总体积约为 `0.279 m³`。
3. 在 FDS 主网格的 `0.1 m` 单元中心执行实体内点判断。
4. 得到约 256 至 258 个有限体积单元，并沿 X 方向合并为约 136 至 138 个 `OBST`。
5. 最终 FDS 座椅体积约为 `0.256 至 0.258 m³`，相对 STL 体积偏差约为 `7.5% 至 8.2%`。
6. 删除原模型中 72 个 `U06.stl` 零厚度壳面记录。
7. 对修改后的完整几何重新执行 DDA 外部热流追踪、受照面生成和探针筛选。

## 探针修正

有限体积几何不能继续使用零厚度壳面的中心点定位方法。探针位置现按真实面坐标加 `0.035 m` 法向偏移计算，并剔除落入任一有限体积 `OBST` 内部的候选面。

最终案例包含 152 个温度探针，其中 `SEAT` 10 个；FDS 预检查已经通过。

## 验证案例

案例：

`Q0400_W0100_az270_el15_H1H7_v5_Qnorm_adapt_HRRtable_seatSTLfill_v1`

固定参数：

| 参数 | 数值 |
|---|---:|
| 光冲量 Q | 400 J/cm² |
| 核爆当量 W | 100 kt |
| 方位角 | 270° |
| 俯仰角 | 15° |
| 终止时间 | 1500 s |
| MPI 进程数 | 32 |
| SEAT HRRPUA | 450 kW/m² |
| BURN_AWAY | `.FALSE.` |

相对表格近似 HRRPUA 对照案例仅改变 U06 座椅几何表达。毁伤阈值、其他材料、燃烧参数和光冲量归一化均未改变。

## 对毁伤评估的影响

该修正可能显著影响结果，不能按“无影响”跳过：

1. 座椅具有真实燃料体积后，聚氨酯泡沫的热容量和燃料质量发生变化。
2. 座椅表面升温速度可能低于零厚度壳面，但点燃后的燃烧持续时间和总放热量可能增加。
3. 座椅火焰对 H1-H7、BED、CURT 等邻近对象的二次加热可能改变。
4. `SEAT` 的峰值温度、阈值持续时间和重度毁伤判定可能改变。
5. 因几何发生变化，该案例必须作为独立几何方案与旧壳面案例对照，不能直接并入原光冲量阈值序列。
