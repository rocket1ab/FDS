# 正常 HRRPUA 光冲量序列

## 目的

在保留高 HRRPUA 上限敏感性案例的前提下，恢复此前采用的正常 HRRPUA，独立运行
Q=50、100、200、300、400 J/cm2 序列。该序列用于评价较保守燃烧强度下的毁伤响应，
不得与高 HRRPUA 上限案例混合形成同一阈值结论。

## 固定参数

W=100 kt，az=270 deg，el=15 deg，T_END=1500 s，采用入射平面光冲量归一化。
几何、审查后厚度、点燃温度、BURN_AWAY、探针、网格和毁伤判据均保持不变。

| 对象 | HRRPUA (kW/m2) |
|---|---:|
| RADM | 75 |
| WINS | 250 |
| BED | 180 |
| CURT | 180 |
| U4 | 100 |
| SEAT | 200 |
| H6 | 200 |
| H7 | 180 |

H1-H5 仍按铝合金 6061 外壳温度判据评估，不设置燃烧 HRRPUA。

## 后台分配

| 节点 | 独立队列顺序 | MPI |
|---|---|---:|
| node03 | Q200 | 32 |
| node04 | Q50，Q300 | 20，32 |
| node05 | Q100，Q400 | 32，32 |

队列仅在节点剩余进程容量和负载均满足要求时启动，每个独立槽同一时刻只运行一个案例。
